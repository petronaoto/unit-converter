Theoretical Background & Specifications Manual

This manual details the exact mathematical models, engineering standards, and explicit stepwise rounding logic utilized across all blocks of the O&G Engineering Converter tool. The diagrams below utilize CSS wireframing to illustrate equations and logical control flows.

Part I: Gas Heating Value & Combustion (JIS K 2301)

Eq 1: Real Gas Fraction Mixing
vi = (xi × Zi) / Σ(xk × Zk)

Volumetric ($v$) to Molar ($x$) state transfer requires specific ideal gas compressibility factors ($Z$) at standard $0^\circ C, 1 atm$.

Eq 2: Heating Value (HHV)
HHVmix = Σ [ round_2(vi × HHVi) ]

Explicit cascade rounding is applied at the intermediate component level to match legacy JIS K 2301 Excel macro outputs.

Eq 3: Specific Gravity (SG)
SGmix = round_3[ Σ(vi × si) ]

Rather than utilizing $MW / 28.96$, SG is calculated directly via summing pure component specific gravities ($s_i$) and forced to 3 decimals.

Eq 4: Wobbe Index (WI)
WI = round_2[ HHVmix / √SGmix ]

The index determines burner interchange capacity. It strictly utilizes the 3-decimal rounded $SG$ from Eq 3 to prevent floating point drift.

Eq 6: Base Combustion Potential
Base MCP = Σ(Si·fi·Ai) / Σ(fi·Ai)

A purely empirical ratio mapping pure combustion speeds ($S_i$) against volumetric fractions ($A_i$).

Eq 7: Final MCP Output
MCP = round_1[ Base MCP × (1 - K) ]

Final combustion speed output metric, truncated to 1 decimal place for standardized regulatory reporting.

Eq 8: Rackett Equation (Volume)
Vi = (R·Tc/Pc) × Zc[1 + (1 - Tr)^(2/7)]

Calculates theoretical pure component saturated liquid molar volume at specific cryogenic conditions.

Eq 9: LNG Liquid Density
ρliq = Σ(xi·MWi) / Σ(xi·Vi)

Assumes ideal volumetric mixing of the individual Rackett molar volumes determined in Eq 8.

Part II: Pipeline Hydraulics & Two-Phase Flow

Eq 11: Darcy-Weisbach
dPfric = fd · (L/D) · (ρ·v² / 2)

Base equation for major frictional losses across a uniform internal diameter pipe.

Eq 12: Static Head Loss
dPstat = ρ · g · Δz

Gravitational pressure loss added linearly to frictional loss. Assumes incompressible flow over segment.

Diagram 13: Colebrook-White Iteration Logic
Assume f = 0.02 ➜ f_new = [ -2 log( ε/3.7D + 2.51/(Re√f) ) ]⁻² ➜ Diff < 1e-6?
Eq 14: Vapor Quality (x)
x = Wv / (Wv + Wl)

Mass fraction of vapor in the flow stream. Core metric for HEM mixing.

Eq 15: HEM Mix Density
1 / ρmix = (x / ρv) + (1-x / ρl)

Homogeneous Equilibrium Model assumes zero slip between gas and liquid phases.

Eq 16: HEM Mix Viscosity
μmix = x·μv + (1-x)·μl

Linear viscosity weighting model for the pseudo-single-phase friction assessment.

Eq 17: Papay Z-Factor
Z = 1 - (3.52P_pr / 100.98T_pr) + (0.274P_pr² / 100.81T_pr)

Empirical natural gas compressibility directly derived from pseudo-reduced T & P limits.

Part III: Flow & Physical Constants

Eq 18: Ideal Gas Law (Vol)
Vact = Vstd × (Tact/273.15) × (101.3/Pact)

Basic expansion ratio from Standard (0°C, 101.325 kPa) to Actual control room conditions.

Eq 19: Standard Volume Reference
scf = Nm³ × 37.3258

Volumetric expansion ratio fixed across the General Tab and Advanced heating modules.

Diagram 20: Bi-Directional Flow Processor
Mass Input [kg/h] ⇋ ρstd ⇋ Volume Output [Nm³]

The GHV module handles volumetric to mass flow dynamically by recalculating Standard Density ($\rho_{std}$) using $MW_{mix}$ any time fraction inputs change.
