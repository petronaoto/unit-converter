Original Quick Introduction (V1.7 Legacy)

The O&G Engineering Converter is a client-side suite built for control room efficiency. It operates via multiple utility tabs: General Tab: Standard unit conversions and the custom module generator. Basic Eng Tab: Pipe Volume and Z-Factor tools. Advanced Tab: Complex, iterative engineering calculators (Compositional GHV, Fanning Hydraulics). Click the dashed button on the General tab to build a new conversion card. Your custom modules are saved locally to your browser.

Theoretical Background & Specifications Manual

This section provides exactly 20 CSS-wireframed diagrams and mathematical proofs for every operation in the O&G Engineering suite, maintaining strict compliance with established standards.

Part I: Gas Compositional Logic (JIS K 2301)

Diagram 1: Real Mixing Mode
vi = (xi·Zi)/Σ(xk·Zk)

Mole ($x$) to Volumetric ($v$) transfer using standard compressibility $Z$.

Diagram 2: Normal Conditions
Normal = 0°C, 1 atm

Normal gas volume (Nm³) is fixed to $0^\circ C$ and $101.325$ kPa.

Diagram 3: Intermediate HHV Rounding
Σ [ round2(vi·HHVi) ]

Matches legacy Excel cascade logic by rounding pure component HHV contributions to 2 decimals before summation.

Diagram 4: Specific Gravity (Air=1)
SG = round3[ Σ(vi·si) ]

SG is calculated via pure component summation and rounded strictly to 3 decimals.

Diagram 5: Wobbe Index (WI)
WI = round2[ HHV / √SG ]

Uses the 3-decimal rounded SG to prevent floating-point drift in interchangeability metrics.

Diagram 6: Inert Attenuation (K)
K = round2[ (ΣA/ΣaiA)·(term1+term2) ]

K-factor evaluation for inerts, forced to 2-decimal percentage precision.

Diagram 7: Base MCP Potential
round2[ Σ(Si·fi·Ai) / Σ(fi·Ai) ]

Combustion potential based on weighted flame speeds, rounded before final evaluation.

Diagram 8: Final MCP Output
MCP = Base MCP · (1 - K)

Output truncated to 1 decimal place per standardized regulatory reports.

Diagram 9: LNG Calibration Engine
ρliq = f( Rackett, Klosek-McKinley )

Evaluates LNG liquid density using ideal volumetric mixing coefficients and precise temperature-dependent intercepts.

Diagram 10: State Calculation Loop
Input➜Normalize➜Eq 1-9

All metrics recalculate instantly upon any compositional change.

Part II: Hydraulics & Gas Laws

Diagram 11: Darcy-Weisbach
dP = fd·(L/D)·(ρv²/2)

Pressure loss model applied across pipe segments.

Diagram 12: Colebrook-White
1/√f = -2log(ε/3.7D + 2.51/Re√f)

Implicit solver for friction factor solved via 100-loop JS iteration.

Diagram 13: Vapor Quality
x = Wv / (Wv + Wl)

Mass fraction of vapor for multi-phase assessment.

Diagram 14: HEM Mixture Density
1/ρmix = x/ρv + (1-x)/ρl

Homogeneous Equilibrium Model assuming zero-slip between phases.

Diagram 15: HEM Viscosity
μmix = x·μv + (1-x)·μl

Weighted mixture viscosity for Reynolds number evaluation.

Diagram 17: Papay Z-Factor
Z = 1 - 3.52Pr/100.98Tr + 0.274Pr²/100.81Tr

Natural gas compressibility estimated from pseudo-reduced properties.

Diagram 18: Expansion Ratio
Vact = Vstd·(Tact/273.15)·(101.3/Pact)

Volume expansion from Standard (Nm³) to Actual (Am³).

Diagram 19: Standard Volume Reference
scf = Nm³ × 37.3258

Fixed volumetric expansion ratio between metric and imperial standards.

Diagram 20: Air Mass Reference
ρair,std = 1.29227 kg/Nm³

Hardcoded air density constant used to link SG to standard gas mass conversion.

Diagram 20: Storage Engine
DOM➜Stringify➜LocalStore

Custom modules are serialized and stored directly in browser memory.
